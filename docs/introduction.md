# 地图基础概念


## 瓦片图

**官方api截图** 

![此处为官方文档中样式控制的截图](../images/api-4.png)  
::: tip

1. 瓦片图可以理解为是一些png、jpg的图片，这些图片放在地图上展示，但是随着地图缩放层级的改变出现放大缩小会产生失真，因此在一些高德地图、百度地图上快速的缩放会出现地图变模糊接着加载一些高清的图片.
2. 常用在加载地图最底层的底图部分.

:::
## 矢量图

::: tip

矢量图一般用作我们业务部分的效果展示，矢量图的操作自由度高，可以按需修改自己想要的样式，达到ui设计稿的标准，处理业务数据.

:::

## 图层
::: tip

图层通常是指位于底图上层的区域，所有的矢量元素，都要放置在一个对应的图层上，在业务处理上，一般将相同业务的元素集成到一个图层统一管理，例如后续示例中创建的面图层、线图层、点图层.

:::
